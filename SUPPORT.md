# Support

## Getting help

- **Documentation** — Check the README and docs in the repository you’re using (e.g. [wabisaby-core](https://github.com/wabisaby/wabisaby-core), [wabisaby-devkit](https://github.com/wabisaby/wabisaby-devkit), [wabisaby-plugin-sdk-go](https://github.com/wabisaby/wabisaby-plugin-sdk-go)).
- **Issues** — For bugs, unclear behavior, or feature requests, open an issue in the relevant repo.
- **Discussions** — For questions, ideas, or how-to help: [Wabi Saby Discussions](https://github.com/orgs/wabisaby/discussions).

## Repositories

| Repo | Use for |
|------|--------|
| [wabisaby-core](https://github.com/wabisaby/wabisaby-core) | Backend, API, plugin runtime, services |
| [wabisaby-devkit](https://github.com/wabisaby/wabisaby-devkit) | DevKit app, local dev setup, submodules |
| [wabisaby-plugin-sdk-go](https://github.com/wabisaby/wabisaby-plugin-sdk-go) | Building plugins in Go |
| [wabisaby-plugins](https://github.com/wabisaby/wabisaby-plugins) | Plugin examples and official plugins |
| [wabisaby-protos](https://github.com/wabisaby/wabisaby-protos) | Protocol buffer definitions |

Open issues and discussions in the repo that best matches your topic.
